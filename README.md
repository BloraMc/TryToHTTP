# TryToHTTP

**TryToHTTP** is an open source Python-based local proxy that helps you connect to the **TryToChill Network** using WebSockets. This means you can still play even if the main server addresses (`mc.trytochill.net` or `eu.trytochill.net`) are not working.

The main reason we made **TryToHTTP** is to help you stay connected to the **TryToChill Network**. It also helps you get around some network problems.

NOTE : Currently, this only works for TryToChill on Java Edition, but i will add Bedrock support to it soon ...

### Features

* **Stay Connected:** **TryToHTTP** gives you another way to connect when the usual addresses are not working.

* **Better Performance:** Sometimes using WebSockets can make your connection more stable or faster.

* ** to Use:** **TryToHTTP** is a simple Python script that does not use a lot of resources.

* **Allowed:** The **TryToChill Network** fully allows the use of **TryToHTTP**.

---

###. Setup

#### 1. Download

Get the version from the **Releases** tab:

* **Easy Version:** Download the `.exe` file. You do not need Python for this.

* **Source Code:** Download the `.py` file. You will need to install some things:

`pip install websockets`

#### 2. Run the Proxy

Open the file. When you see the message that says it is ready **TryToHTTP** is working.

#### 3. Join the Server

Open Minecraft. Connect to this address:

`127.0.0.1:25565`

*Now **TryToHTTP** will send your traffic to the **TryToChill Network** using WebSockets.*

---

###. Notes

* **Keep it Open:** You need to keep the **TryToHTTP** window open while you are playing.

* **Check the Address:** Make sure no other programs are using the address `25565` or **TryToHTTP** will not work.

* **Compare:** If your connection is slow try connecting to the main addresses to see which way is better, for you.
